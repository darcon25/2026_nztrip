# 2026 NZtrip 專案

紐西蘭南島家族旅遊網站（React + Vite + Express + SQLite，資料來自 Google Sheet）。

## 改版計劃與進度
本專案正在進行手機優先的露營風單頁式改版。
- **接手先讀 `docs/進度.md`**（目前進度、決策、待辦、踩過的坑）。
- 完整設計見 `docs/改版計劃.md`；Sheet 欄位範本見 `docs/Sheet欄位範本.md`。

## 開發指令
- `npm run dev` — 啟動開發伺服器（port 3000）
- `npm run lint` — 型別檢查（tsc --noEmit）
- `npm run build` — 建置
