## 環境/驗證備忘
- 啟動：`cd ~/Projects/2026_nztrip && npm run dev`（port 3000）。lint：`npm run lint`。
- **Claude in Chrome 的視窗縮放無法真正模擬手機寬度**（innerWidth 卡在 1281），截圖一律桌機版。要看手機版請用瀏覽器 DevTools 裝置模式；手機 RWD 在程式裡是對的（亮點卡片手機一欄）。註：把 `body.style.width` 改成 390px 也**無效**，Tailwind 的 `sm:` 是 media query 看 viewport；要驗手機版得暫時移掉 `sm:` class。
- **分頁在背景時地圖標記會「消失」不是 bug**：標記用 motion 進場動畫，Chrome 對背景分頁節流 rAF，動畫會凍結在半透明狀態。截圖前確認 `document.visibilityState === 'visible'`。
- 每次改完務必 `npm run lint` 再 commit。
