## 重要決策（別再重議）
- 家庭身分**用「名稱」**，不用 id 欄（使用者定案；Sheet 預算分頁 id 欄標題空白不可靠）。
- 分帳資料存**網站後端 SQLite**，不寫 Google Sheet（零設定、可稽核）。
- 行程**以 Google Sheet 為準**；互動地圖照 Sheet 畫，美術稿排程差異不影響網站。
- 分帳分攤**以家庭為單位、統一 NZD**。
